# Integrations

SparkClub integrates with standard libraries to provide rich functionality.

## Reporting Engines
* **jsPDF**: Generates dynamic client-side PDF statements, receipts, and invoices with visual layouts.
* **SheetJS (XLSX)**: Converts JSON transactional ledgers into fully styled Excel documents.

## API Service Wrappers
* **Anthropic Messages API**: Powers the AI assistant and recommendation panel.
