# User Roles

SparkClub supports multiple operational roles:

* **Super Admin**: High-level system settings, multi-tenant creation, and global health check analytics.
* **Faculty Coordinator**: Assigned institutional observer who reviews events, verifies compliance, and approves funds.
* **Club Head**: Runs day-to-day operations, creates events, tracks tasks, and assigns sub-departments.
* **Treasurer**: Bookkeeper who manages ledger accounts, logs transactions, and maintains fiscal categories.
* **Student Member**: Participates in events, updates tasks, and submits reimbursement claims.
