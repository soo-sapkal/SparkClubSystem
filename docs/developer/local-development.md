# Local Development

Run the frontend and backend simultaneously in development.

## Run Servers
* **Start Backend**: In `sparkclub/backend`, run:
  ```bash
  npm run dev
  ```
  Backend starts on `http://localhost:3001`.
* **Start Frontend**: In `sparkclub/frontend`, run:
  ```bash
  npm run dev
  ```
  Frontend starts on `http://localhost:5173`.
