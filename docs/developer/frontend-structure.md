# Frontend Structure

Inside `sparkclub/frontend/src`:

* **`components/`**: Form fields, card elements, layout wrapper, sidebars.
* **`context/`**: `AuthContext.jsx` holds user roles and active sessions.
* **`pages/`**: SuperAdmin, Faculty, ClubHead, Treasurer, and Student dashboards.
* **`services/`**: `api.js` contains axios endpoints wrapper.
