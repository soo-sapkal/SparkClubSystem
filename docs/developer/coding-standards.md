# Coding Standards

We follow these guidelines:

## Code Rules
* **Javascript Style**: Use ES6 modules and async/await blocks.
* **REST Constraints**: Return HTTP `400` on invalid requests, `401`/`403` on permissions error, and `200`/`201` on successful action.
* **Tailwind CSS styling**: Maintain semantic color standards (e.g., `text-slate-400`, `bg-slate-950`).
