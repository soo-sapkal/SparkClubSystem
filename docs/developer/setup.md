# Setup & Installation

Follow these steps to spin up SparkClub locally.

## Requirements
* **Node.js**: v18 or later
* **npm**: v9 or later
* **SQLite**: v3.x (optional, database is embedded)

## Setup Steps
1. Clone the repository.
2. Initialize backend:
   ```bash
   cd sparkclub/backend
   npm install
   ```
3. Initialize frontend:
   ```bash
   cd ../frontend
   npm install
   ```
